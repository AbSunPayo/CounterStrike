1. Clique no botão "Files" no canto superior direito
2. Baixe: cs2_skin_monitor_github.tar.gz
3. Descompacte no seu computador
4. Siga as instruções no README.md
5. Configure o .env com suas credenciais
6. Envie para o GitHub usando git

   tar -xzf cs2_skin_monitor_github.tar.gz
   cd cs2_skin_monitor_github
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/SEU_USUARIO/cs2-skin-monitor.git
   git push -u origin main
